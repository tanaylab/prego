# prego: PWM Regression Optimizer

Implements regression algorithms for finding motifs in DNA. It contains
algorithms which were described at: "Extensive low-affinity
transcriptional interactions in the yeast genome"
[doi:10.1101/gr.5113606](https://doi.org/10.1101/gr.5113606) .
