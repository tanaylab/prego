# PSSMs from the HOMER motif database

PSSMs from the HOMER motif database

## Usage

``` r
HOMER_motifs
```

## Format

A data frame containing the PSSMs ('A', 'C', 'G' and 'T' columns), with
an additional column 'motif' containing the motif name.

## Source

<http://homer.ucsd.edu/homer/motif/>
