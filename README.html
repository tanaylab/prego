prego

The prego package implements simple regression algorithms for finding motifs in DNA. You can either use it to find motif which are discriminating between two or more clusters of DNA sequences, or for generting motifs from one or more continuous variables.

The PREGO algorithm is described here.

Installation

You can install the development version of prego like so:

remotes::install_github("tanaylab/prego")

Usage

library(prego)
